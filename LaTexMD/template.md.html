# This is the LaTex template The stuff below will make sure the markdown gets transformed to viewable markdown in your browser. This markdown flavor has a few differences from GitHub markdown: * Nested lists don't have to be aligned, 2 spaces are enough. * To insert a single newline, use <br/> instead of 2 spaces and a return. * To insert raw HTML as inline code use raw code tags with escaped greater/smaller than brackets. * To insert raw HTML as code between 3 backticks, escape greater/smaller than brackets.
The escapes are `&lt;` for < and `&gt;` for >.
